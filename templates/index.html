Complete Setup Guide
1
Python & Dependencies
Requires Python 3.9 or higher.
# Clone or extract the toolkit cd cloud-osint-v3 # Create virtual environment (recommended) python3 -m venv venv source venv/bin/activate # Linux/Mac venv\Scripts\activate # Windows # Install all dependencies pip install -r requirements.txt
requirements.txt includes: flask, requests, dnspython, boto3
2
Run the App
python3 app.py # → Open http://127.0.0.1:5000
Or use the provided run script:
chmod +x run.sh && ./run.sh
3
GitHub Token (Free)
Without a token: 10 requests/hour limit. With token: 5,000/hour.
1. Go to: github.com/settings/tokens 2. Click "Generate new token (classic)" 3. Select ONLY: public_repo (read-only) 4. Copy the ghp_xxxx token → paste in sidebar
4
Shodan API Key ($)
Shodan has a free tier (100 results/month). Membership plan ~$65/yr for full access.
1. Register at: shodan.io 2. Go to: shodan.io/dashboard 3. Copy "API Key" → paste in sidebar 4. Free tier: use "shodan.io/api-plan" for limits
Free accounts: limited to 1 API result page. Still useful for basic scanning.
5
Censys API (Free tier)
Censys offers a free community tier with 250 queries/month.
1. Register at: censys.io 2. Go to: censys.io/account/api 3. Copy "API ID" and "API Secret" 4. Paste both into the sidebar fields
6
VirusTotal API (Free)
Free tier: 4 lookups/min, 500/day. Enough for passive DNS subdomain discovery.
1. Register at: virustotal.com 2. Go to: virustotal.com/gui/my-apikey 3. Copy the API key → paste in sidebar
7
AWS IAM Testing
⚠ ONLY use this on keys you found via OSINT on systems you own or are authorized to test.
# If you found a leaked AWS key via GitHub scanning: AWS Access Key: AKIAIOSFODNN7EXAMPLE AWS Secret Key: wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY # The tool calls STS GetCallerIdentity to verify # If valid: shows Account ID + ARN + lists S3 buckets # Requires: pip install boto3
boto3 is in requirements.txt. If you skip this module it simply reports "no credentials".
8
No-Key Passive Mode
Without ANY API keys, these modules still work 100%:
✓ CRT.sh — certificate transparency (free, no key) ✓ Wayback Machine — CDX API (free, no key) ✓ HackerTarget — host search (free, no key) ✓ DNS brute-force — 80+ subdomain wordlist ✓ Cloud Storage — AWS/Azure/GCP HTTP probes ✓ Firebase probe — public RTDB detection ✓ Azure AD — OpenID tenant discovery
This is already more than most commercial tools offer for free.
